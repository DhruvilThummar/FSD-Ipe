Unit 4: Introduction to CSS

4.1 CSS Basics & Selectors

CSS Types:

  • Inline CSS - style attribute
  • Internal CSS - <style> tag in <head>
  • External CSS - separate .css file with <link> tag

CSS Selectors:

  • Element Selector: p, div, h1
  • ID Selector: #myId
  • Class Selector: .myClass
  • Attribute Selector: [type="text"]
  • Universal Selector: *
  • Descendant Selector: div p
  • Child Selector: div > p
  • Grouping Selector: h1, h2, h3
  • Pseudo-class: :hover, :focus, :nth-child()
  • Pseudo-element: ::before, ::after, ::first-line

CSS Specificity: ID > Class > Element

4.2 Background, Text & Font Properties

Background Properties:

  • background-color, background-image, background-repeat
  • background-position, background-size, background-attachment
  • background-clip, background (shorthand)

Text Properties:

  • color, text-align, text-indent, text-transform
  • text-decoration, letter-spacing, word-spacing
  • line-height, text-shadow

Font Properties:

  • font-family, font-size, font-style, font-weight
  • font-variant, font (shorthand)
  • Google Fonts via <link> or @import

4.3 Box Model & Layout

Box Model Properties:

  • margin, padding, width, height
  • border-style, border-width, border-color, border
  • box-sizing (content-box, border-box)
  • box-shadow

Display Properties:

  • inline, block, inline-block, none
  • flex, grid

Flexbox Properties:

  • display: flex; flex-direction (row, column)
  • justify-content, align-items, flex-wrap, gap
  • order, flex (shorthand)

Grid Layout Properties:

  • display: grid; grid-template-columns, grid-template-rows
  • gap (grid-gap), grid-column, grid-row

Positioning:

  • position: static, relative, absolute, fixed, sticky
  • top, right, bottom, left, z-index
