üìå Essential Topics Checklist

Unit 1: Introduction to Web & Web Design 5%

  • WWW (World Wide Web) basics
  • HTTP protocol and request-response cycle
  • HTTP status codes (200, 404, 500, 301, etc.)
  • URL and Client-Server architecture
  • Website planning and design principles
  • Browser compatibility and responsive design
  • Navigation design and user experience

Unit 2: HTML Structure & Core Tags 10%

  • Basic HTML structure (DOCTYPE, html, head, body)
  • Meta tags (charset, viewport, description, keywords)
  • Semantic HTML5 tags (header, footer, nav, section, article, aside)
  • Text formatting tags (strong, em, mark, del, ins, sub, sup)
  • Heading tags (h1-h6) and paragraph tags
  • Image tags with attributes
  • Hyperlinks and anchor tags
  • Lists (ul, ol, li with type attribute)
  • Tables (table, tr, td, th, colspan, rowspan)
  • Semantic vs non-semantic tags

Unit 3: Forms, Graphics & Interactive Elements 10%

  • Form structure and form elements
  • Input types (text, password, email, number, checkbox, radio, date, type, file, color, range)
  • Form attributes (action, method, name, id, value, placeholder, required)
  • Fieldset and legend for grouping
  • Select, option, optgroup dropdown menus
  • Textarea for multi-line input
  • SVG basics (rect, circle, line, polygon, path)
  • SVG drawing and styling
  • Image maps (map and area tags)
  • Datalist for predefined input options

Unit 4: Introduction to CSS 10%

  • CSS types (inline, internal, external)
  • CSS selectors (element, ID, class, attribute, universal, descendant, child, grouping)
  • Pseudo-classes and pseudo-elements
  • CSS Specificity (ID > Class > Element)
  • Background properties (color, image, repeat, position, size, attachment)
  • Text properties (color, align, indent, transform, decoration, shadow)
  • Font properties (family, size, style, weight, variant, Google Fonts)
  • Box model (margin, padding, border, width, height, box-sizing)
  • Display (inline, block, inline-block, flex, grid, none)
  • Flexbox (direction, justify-content, align-items, flex-wrap, gap, order)
  • CSS Grid (grid-template-columns, grid-template-rows, gap, grid-column, grid-row)
  • Positioning (static, relative, absolute, fixed, sticky, z-index)

Unit 5: CSS Effects, Animations & Responsive 10%

  • CSS columns (column-count, column-gap, column-rule-style)
  • Responsive images (rounded, thumbnail, transparent)
  • Gradients (linear, radial, conic)
  • Transformations (rotate, scale, translate, skew, 3D transforms)
  • Transitions (property, duration, timing-function, delay)
  • Animations (@keyframes, animation-name, animation-duration, animation-timing-function)
  • Animation iteration count and direction
  • CSS variables (:root, var() function)
  • Media queries (syntax, media types, breakpoints)
  • Responsive design principles (mobile-first approach)

Unit 6: Introduction to Bootstrap & Utility Classes 10%

  • Bootstrap setup (CDN and file inclusion)
  • Container types (fixed and fluid)
  • Grid system (12-column, row, col classes)
  • Responsive prefixes (col-sm, col-md, col-lg, col-xl, col-xxl)
  • Utility classes (spacing, display, colors, sizing, borders)
  • Bootstrap components (buttons, alerts, badges, spinners, images)
  • Bootstrap tables and icons
  • Customizing Bootstrap

Unit 7: Bootstrap Components - Navigation & Forms 15%

  • Navbar (navbar-brand, navbar-nav, navbar-expand-lg)
  • Dropdown menus in navbar
  • Collapsible navbar for mobile
  • Offcanvas sidebar (offcanvas-start, offcanvas-end)
  • Carousel/slideshow components
  • Modal dialogs
  • Scrollspy functionality
  • Bootstrap forms (form-control, form-label)
  • Floating labels in forms
  • Form validation (is-valid, is-invalid)
  • Pagination component
  • Checkboxes and radio buttons in Bootstrap

Unit 8: Core JavaScript Concepts & Functions 5%

  • Variables (var, let, const) and scoping
  • Data types (number, string, boolean, null, undefined, object, symbol)
  • Arrays and array methods (push, pop, map, filter, forEach, includes, find)
  • Objects and object properties
  • Template literals and string interpolation
  • Loops (for, while, do-while, for-of, for-in)
  • Conditional statements (if-else, switch-case)
  • Functions (declaration, expression, arrow functions)
  • Function parameters and return values
  • Default parameters
  • Rest parameter and spread operator
  • Popup boxes (alert, confirm, prompt)

Unit 9: JavaScript DOM Manipulation & Events 15%

  • DOM structure and document object
  • Element selection (getElementById, getElementsByClassName, querySelector, querySelectorAll)
  • innerHTML, innerText, textContent properties
  • Attribute methods (setAttribute, getAttribute, removeAttribute)
  • Style manipulation (element.style properties)
  • Class manipulation (classList.add, remove, toggle)
  • Creating and removing elements
  • Mouse events (click, double-click, mouseover, mouseout, mouseenter, mousemove)
  • Keyboard events (keydown, keyup, keypress)
  • Form events (input, submit, change, focus, blur)
  • Window events (load, scroll, resize)
  • addEventListener method
  • Event object properties (target, type, key)
  • preventDefault() and stopPropagation()

Unit 10: Modern JavaScript - Async, Fetch & Forms 10%

  • Form validation (HTML5 and JavaScript)
  • Regular expressions for pattern matching
  • Form processing and data collection
  • localStorage and sessionStorage
  • Promises (.then(), .catch(), .finally())
  • Promise states (pending, fulfilled, rejected)
  • Async functions and await keyword
  • Try-catch for error handling
  • Fetch API for HTTP requests
  • GET and POST requests
  • JSON parsing (JSON.stringify, JSON.parse)
  • Error handling in async operations

‚ö° Quick Reference Code Snippets

JavaScript Selectors

// Modern selectors (recommended) const el1 = document.querySelector('#myId'); const el2 = document.querySelectorAll('.myClass'); const el3 = document.querySelectorAll('[type="text"]'); // Legacy selectors (still used) const el4 = document.getElementById('myId'); const el5 = document.getElementsByClassName('myClass');

Event Handling

// addEventListener (preferred) element.addEventListener('click', function(event) { console.log(event.target); console.log(event.type); }); // Arrow function button.addEventListener('click', () => { element.style.color = 'red'; });

Flexbox Example

.flex-container { display: flex; flex-direction: row; justify-content: space-around; /* horizontal */ align-items: center; /* vertical */ flex-wrap: wrap; gap: 20px; }

Media Query

/* Mobile-first approach */ .container { width: 100%; } @media (min-width: 768px) { .container { width: 750px; } } @media (min-width: 992px) { .container { width: 970px; } }

Async/Await Example

async function getData() { try { const response = await fetch('https://api.example.com/data'); const data = await response.json(); console.log(data); } catch (error) { console.error('Error:', error); } }
