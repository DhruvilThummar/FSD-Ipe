Unit 6: Introduction to Bootstrap & Utility Classes

6.1 Bootstrap Setup & Containers

Bootstrap Setup:

  • Include Bootstrap CDN in <head> section
  • Include JS file before closing </body>
  • Bootstrap 5 requires no jQuery

Containers:

  • .container - Fixed width container
  • .container-fluid - Full width container
  • Responsive breakpoints: xs, sm, md, lg, xl, xxl

Utility Classes:

  • Spacing: m, p, mt, mb, ml, mr, pt, pb, pl, pr (margin, padding)
  • Display: d-none, d-block, d-flex, d-grid
  • Colors: text-primary, bg-danger, border-success
  • Sizing: w-25, h-50, w-100
  • Borders: border, border-top, rounded, rounded-circle
  • Positioning: position-absolute, position-fixed
  • Typography: text-center, text-start, fs-1 to fs-6
  • Shadows: shadow, shadow-sm, shadow-lg
  • Flex: flex-direction, justify-content, align-items, gap

6.2 Grid System & Components

Grid System:

  • 12-column grid system
  • Classes: row, col, col-md-6, col-lg-4
  • Responsive: col-sm-12 col-md-6 col-lg-4
  • Offset columns: offset-md-2

Bootstrap Components:

  • Alerts: .alert, .alert-success, .alert-danger, alert-dismissible
  • Buttons: .btn, .btn-primary, .btn-lg, .btn-disabled
  • Spinner: .spinner-border, .spinner-grow (loading indicators)
  • Images: .img-fluid, .rounded, .img-thumbnail
  • Icons: Bootstrap Icons library
  • Tables: .table, .table-striped, .table-hover, .table-dark

6.3 Advanced Components

Components to Learn:

  • Dropdown: .dropdown, .dropdown-menu, data-bs-toggle
  • Cards: .card, .card-body, .card-title, .card-text
  • Collapse: .collapse, data-bs-toggle="collapse"
  • Accordion: .accordion, .accordion-item, accordion-button
