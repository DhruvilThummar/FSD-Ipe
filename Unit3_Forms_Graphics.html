Unit 3: Forms, Graphics & Interactive Elements

3.1 HTML Forms & Input Elements

Form Tags:

  • <form> - Container for form elements
  • <fieldset> <legend> - Group related form elements
  • <label> - Text for input fields
  • <input> - Various input types (text, password, email, number, checkbox, radio, date, file, etc.)
  • <textarea> - Multi-line text input
  • <select> <option> <optgroup> - Dropdown lists
  • <button> - Clickable button
  • <datalist> - Pre-defined options for input

Important Input Attributes:

  • type, name, id, value, placeholder
  • required, readonly, disabled, checked
  • maxlength, minlength, step, min, max
  • autocomplete, autofocus, multiple, accept

3.2 SVG (Scalable Vector Graphics)

SVG Elements:

  • <svg> - Container for SVG
  • <rect> - Rectangles
  • <circle> - Circles
  • <line> - Lines
  • <polygon> - Polygons (stars, shapes)
  • <path> - Custom paths
  • <text> - Text inside SVG

3.3 HTML Image Maps

Image Maps:

  • <map> - Define clickable regions
  • <area> - Define clickable areas with href
  • Shapes: rect, circle, poly
  • Coordinates for hotspots
