Practical Projects & Hands-On Exercises

Project 1: Bootstrap Offcanvas Sidebar

Create a webpage with a button that toggles an offcanvas sidebar. Include header with close button, body with navigation links. Links to: Unit 2, 3

Project 2: Mouse Events Handler

Handle mouse events: heading turns yellow on hover (black on leave). Buttons to show time/date, change background color to red/green. Links to: Unit 4, 5

Project 3: Contact Form with LocalStorage

Create form for username, email, message. Save to localStorage on submit. Retrieve and display stored data below form. Links to: Unit 6, 7

Project 4: Advanced Layout Design

Design responsive layout with multiple sections, cards, and interactive elements. Showcase responsive grid system and Bootstrap components. Links to: All Units

Project 5: E-Commerce Product Page

Build page with product cards, filters, shopping cart functionality using JavaScript and localStorage. Implement form validation and async operations.

Project 6: Interactive Dashboard

Create dashboard with charts, data from API (Fetch), form inputs, and dynamic content updates. Demonstrate async operations and DOM manipulation.

Key Concepts to Master

  • CSS Specificity: ID > Class > Element. Remember importance of proper selector usage.
  • Flexbox vs Grid: Flexbox for 1D layouts, Grid for 2D layouts. Practice both extensively.
  • ES6 Features: Arrow functions, destructuring, spread operator, template literals.
  • DOM Methods: querySelector > getElementById/getElementByClass for modern JS.
  • Promises vs Async/Await: Both are important; async/await is cleaner syntax.
  • Responsive Design: Mobile-first approach with media queries at 576px, 768px, 992px.
  • Bootstrap Grid: 12-column system; master col-sm, col-md, col-lg classes.
  • Form Validation: HTML5 validation + JavaScript validation for robust forms.

Exam Preparation Tips

  • Practice all 24+ practicals provided in syllabus thoroughly
  • Create your own projects combining multiple units
  • Focus on Bootstrap and JavaScript as weightage is higher (30% combined)
  • Understand WHY things work, not just HOW to code them
  • Test your code on different browsers and screen sizes
  • Write clean, commented code with proper indentation
  • Use developer tools (F12) to debug and inspect elements
  • Create a cheat sheet for common CSS and JS patterns
  • Practice time management - complete practicals within time limit
  • Focus on responsive design - most exams emphasize mobile compatibility
